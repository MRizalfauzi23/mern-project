# MERN Project

Recruitment management system (jobs, applications, pipeline, admin analytics) with a public applicant form.

## Scripts

**Backend**
- `npm run dev` (in `backend/`)
- `npm test` (in `backend/`)

**Frontend**
- `npm run dev` (in `frontend/`)
- `npm run build` (in `frontend/`)

## CI
GitHub Actions workflow builds frontend and runs backend tests on pushes and PRs to `main`.
